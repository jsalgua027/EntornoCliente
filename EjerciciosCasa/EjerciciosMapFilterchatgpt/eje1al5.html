Ejercicio 1

Ejercicio 1 (uso de map): Enunciado: Dado un array de números, utiliza la función map para crear un nuevo array que contenga el doble de cada número.

Ejercicio 2 (uso de filter): Enunciado: Dado un array de números, utiliza la función filter para crear un nuevo array que contenga solo los números pares.

Ejercicio 3 (uso de find): Enunciado: Dado un array de objetos que representan libros, cada objeto tiene propiedades como "título" y "autor". Utiliza la función find para encontrar el primer libro escrito por un autor específico.

Ejercicio 4 (combinando map y filter): Enunciado: Dado un array de personas, cada persona es un objeto con propiedades como "nombre" y "edad". Utiliza map para crear un nuevo array que contenga solo los nombres de las personas mayores de 30 años.
